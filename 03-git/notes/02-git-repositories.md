# Git Repositories

## What Is It?

A Git repository stores a project's files together with its complete version history.

Repositories can exist locally on a computer and remotely on platforms such as GitHub.

---

## Why Is It Useful?

Repositories provide a central location for managing source code and tracking changes throughout a project's lifetime.

---

## My Experience

My DevOps_Journey repository became the central place for all my documentation. Every note, lab and project I completed was committed locally and then pushed to GitHub, allowing me to continue working from both my MacBook and work laptop.

---

## Best Practices

- Keep repositories organised.
- Use a clear folder structure.
- Commit frequently.
