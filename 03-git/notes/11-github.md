# GitHub

## What Is It?

GitHub is a cloud-based platform that hosts Git repositories.

It allows developers to store projects remotely, collaborate with others and maintain version history.

---

## Why Is It Useful?

GitHub provides a central location for repositories, making it easy to work across multiple devices and share code with other developers.

---

## My Experience

Throughout this DevOps Journey, GitHub became the central location for all of my work.

Whether I was documenting Linux on my work laptop or completing Bash projects on my MacBook, GitHub allowed both machines to stay synchronised.

It also became the public portfolio showcasing my progress as I moved through my DevOps roadmap.

---

## Best Practices

- Push work regularly.
- Keep repositories organised.
- Write meaningful README files.
- Use GitHub as your source of truth.
