# Lab 01 - Repository Initialisation

## Objective

Create and initialise a Git repository.

---

## Environment

- macOS
- Ubuntu 24.04 (WSL)
- GitHub

---

## Commands Used

```bash
git init
```

---

## Tasks Completed

- Initialised a local Git repository.
- Understood the purpose of the `.git` directory.

---

## Challenges

Understanding that Git stores repository metadata separately from project files.

---

## Result

Successfully created a local Git repository.

---

## What I Learned

Every Git project begins with a repository that tracks the complete history of the project.

---

## Real-World Relevance

Every software project starts by creating or cloning a Git repository.
