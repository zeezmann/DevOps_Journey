# Lab 08 - Push and Pull

## Objective

Synchronise local and remote repositories.

---

## Environment

- macOS
- Ubuntu 24.04 (WSL)

---

## Commands Used

```bash
git push
git pull
```

---

## Tasks Completed

- Pushed commits to GitHub.
- Pulled changes onto another machine.

---

## Challenges

Understanding that commits remain local until pushed.

---

## Result

Successfully synchronised my work laptop and MacBook.

---

## What I Learned

GitHub acts as the central repository between multiple computers.

---

## Real-World Relevance

Developers regularly push and pull while working across different machines.
