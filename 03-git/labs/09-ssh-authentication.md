# Lab 09 - SSH Authentication

## Objective

Configure SSH authentication for GitHub.

---

## Environment

- macOS
- Ubuntu 24.04 (WSL)

---

## Commands Used

```bash
ssh-keygen
ssh -T git@github.com
```

---

## Tasks Completed

- Generated SSH keys.
- Added public keys to GitHub.
- Verified authentication.

---

## Challenges

Understanding public and private key pairs.

---

## Result

Successfully authenticated using SSH.

---

## What I Learned

SSH provides secure authentication without repeatedly entering credentials.

---

## Real-World Relevance

SSH authentication is the preferred method for professional Git workflows.
