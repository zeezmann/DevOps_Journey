# Lab 03 - Tracking Changes

## Objective

Track modifications made to files using Git.

---

## Environment

- macOS
- Ubuntu 24.04 (WSL)

---

## Commands Used

```bash
git status
git diff
```

---

## Tasks Completed

- Modified files.
- Checked repository status.
- Reviewed changes before committing.

---

## Challenges

Understanding the difference between modified and staged files.

---

## Result

Successfully tracked file changes throughout development.

---

## What I Learned

Checking changes before committing became an important part of my workflow.

---

## Real-World Relevance

Developers constantly monitor repository status while working.
