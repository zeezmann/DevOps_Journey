# Lab 02 - Cloning Repositories

## Objective

Clone an existing Git repository from GitHub.

---

## Environment

- macOS
- Ubuntu 24.04 (WSL)

---

## Commands Used

```bash
git clone
```

---

## Tasks Completed

- Cloned an existing repository.
- Verified the remote repository.

---

## Challenges

Choosing SSH over HTTPS for authentication.

---

## Result

Successfully cloned a repository onto multiple machines.

---

## What I Learned

Cloning copies both the project files and the complete Git history.

---

## Real-World Relevance

Developers clone repositories before contributing to projects.
