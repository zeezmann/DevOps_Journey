# Lab 04 - Commits

## Objective

Create meaningful commits to record project progress.

---

## Environment

- macOS
- Ubuntu 24.04 (WSL)

---

## Commands Used

```bash
git commit -m
```

---

## Tasks Completed

- Created commits with meaningful messages.
- Grouped related work into single commits.

---

## Challenges

Learning to avoid unnecessary commits.

---

## Result

Built a cleaner and easier-to-follow commit history.

---

## What I Learned

A commit should represent one logical piece of work.

---

## Real-World Relevance

Clean commit history makes projects easier to review and maintain.
