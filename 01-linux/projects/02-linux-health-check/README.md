# Linux Health Check

## Overview

The Linux Health Check is a Bash utility that performs a quick health assessment of a Linux system.

It checks common system resources and reports whether they appear healthy.

---

## Features

- Check disk usage
- Check memory usage
- Check internet connectivity
- Check whether Git is installed
- Check whether SSH is installed

---

- Ubuntu 24.04 (WSL)
