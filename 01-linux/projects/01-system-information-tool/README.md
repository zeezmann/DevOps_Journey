# Linux System Information Tool

## Overview

The Linux System Information Tool is a Bash application that displays useful information about the current Linux system in a clean, readable format.

The goal of this project is to combine Linux and Bash concepts into a practical utility that could be useful during system administration and troubleshooting.

---

## Features

- Display current user
- Display hostname
- Display operating system
- Display kernel version
- Display uptime
- Display current date and time
- Display current working directory
- Display memory usage
- Display disk usage
- Display IP address

## How to Run

Make the script executable.

```bash
chmod +x system-info.sh
```

Run the script.

```bash
./system-info.sh
```
