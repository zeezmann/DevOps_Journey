# Linux Labs

This directory contains the practical Linux labs I completed while studying Linux Fundamentals.

The purpose of these labs is to reinforce the concepts covered in my notes through hands-on practice.

Rather than simply following commands, each lab documents what I actually did, the environment I used, any challenges I encountered and what I learned from the exercise.

---

## Lab Structure

Each lab follows the same structure:

- Objective
- Environment
- Commands Used
- Tasks Completed
- Challenges
- Result
- What I Learned
