# Lab 01 - File Navigation

## Objective

Become comfortable navigating the Linux file system and performing common file management tasks using the command line.

---

## Environment

- Ubuntu (VMware Fusion)
- Ubuntu 24.04 (WSL)
- macOS Terminal

---

## Commands Used

```bash
pwd
ls
ls -lah
cd
mkdir
touch
cp
mv
rm
find
```

---

## Tasks Completed

- Navigated between directories using relative and absolute paths.
- Created new directories with `mkdir`.
- Created files using `touch`.
- Listed files using various `ls` options.
- Copied and renamed files.
- Removed files and directories.
- Located files using the `find` command.

---

## Challenges

When I first started learning Linux, I occasionally lost track of my current location in the file system.

Using `pwd` regularly and making use of tab completion helped me become much more confident navigating directories.

---

## Result

Successfully completed common Linux file management tasks and became comfortable navigating the Linux file system from the terminal.

---

## What I Learned

Understanding the Linux file system and practising navigation regularly made using the terminal feel much more natural. I also realised that knowing where I am before running commands helps avoid unnecessary mistakes.
