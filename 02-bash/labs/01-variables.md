# Lab 01 - Variables

## Objective

Learn how to create and use variables in Bash scripts.

---

## Environment

- Ubuntu 24.04 (WSL)
- macOS Terminal

---

## Commands Used

```bash
name="Azeez"
echo "$name"
```

---

## Tasks Completed

- Created variables.
- Displayed variable values.
- Used variables throughout scripts.

---

## Challenges

Remembering to avoid spaces around the `=` operator.

---

## Result

Successfully stored and reused values within Bash scripts.

---

## What I Learned

Variables make scripts easier to maintain by storing values that can be reused throughout the script.

---

## Real-World Relevance

Variables are used extensively in automation scripts for usernames, directories, file paths and configuration values.
