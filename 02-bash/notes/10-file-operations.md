# File Operations

Bash provides powerful commands for creating, copying, moving and deleting files and directories.

These commands are commonly used in automation scripts and system administration.

---

## Common Commands

- `touch`
- `cp`
- `mv`
- `rm`
- `mkdir`
- `rmdir`
- `cat`

---

## Why File Operations Matter

Most Bash scripts interact with files in some way, whether creating backups, moving log files or generating reports.

---

## Real-World Example

Examples include:

- Creating backup files
- Organising directories
- Moving log files
- Deleting temporary files

---

## What I Learned

Understanding file operations allowed me to automate repetitive file management tasks instead of performing them manually.
