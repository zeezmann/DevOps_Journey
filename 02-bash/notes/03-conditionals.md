# Conditional Statements

Conditional statements allow a script to make decisions based on whether a condition is true or false.

---

## Syntax

```bash
if [ condition ]; then

    commands

fi
```

---

## Common Operators

- `-eq`
- `-ne`
- `-gt`
- `-lt`
- `-ge`
- `-le`

String operators:

- `=`
- `!=`
- `-z`
- `-n`

---

## Why Conditionals Matter

Conditional logic allows scripts to react differently depending on user input, command output or system state.

---

## Real-World Example

Examples include:

- Checking whether a file exists.
- Validating user input.
- Verifying commands completed successfully.

---

## What I Learned

Conditionals are the decision-making component of Bash and are essential for creating useful automation scripts.
