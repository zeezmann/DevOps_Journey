# Functions

Functions group related commands together so they can be reused throughout a script.

Using functions makes scripts easier to read, organise and maintain.

---

## Syntax

```bash
greet() {

    echo "Hello"

}
```

Calling the function:

```bash
greet
```

---

## Why Functions Matter

Functions prevent duplicated code and allow scripts to be broken into logical sections.

---

## Real-World Example

Functions can be used for:

- Creating backups
- Validating input
- Displaying menus
- Logging information
- Printing reports

---

## What I Learned

Functions made my scripts feel much more organised.

Breaking a script into small reusable sections makes troubleshooting and future improvements much easier.
