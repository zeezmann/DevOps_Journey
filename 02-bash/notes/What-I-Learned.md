# What I Learned

Completing the Bash module gave me a much better understanding of how automation works in Linux.

At the beginning, Bash scripts felt like a collection of commands. As I progressed, I learned how variables, conditionals, loops, functions and command substitution work together to create practical automation tools.

The biggest lesson I learned was to think about the structure of a script before writing code. Breaking scripts into small, reusable functions made them much easier to understand, maintain and troubleshoot.

Building projects throughout this module reinforced the importance of writing clear, well-documented scripts rather than simply making them work.

Overall, Bash has given me a strong foundation for Linux administration, automation and future DevOps work.
