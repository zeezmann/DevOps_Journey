# Log File Analyzer

## Overview

This project analyses Linux log files by searching for keywords supplied by the user.

It demonstrates text processing and file handling using Bash.

---

## Features

- Prompt for log file
- Prompt for search keyword
- Display matching lines

---

## Technologies Used

- Bash
- grep

---

## Skills Demonstrated

- User Input
- Variables
- grep
- Conditional Statements
- File Validation
