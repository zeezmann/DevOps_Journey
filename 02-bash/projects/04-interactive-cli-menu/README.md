# Interactive CLI Menu

## Overview

The Interactive CLI Menu is a Bash application that provides users with a simple text-based menu for executing common Linux commands.

It demonstrates loops, functions, case statements and user interaction.

---

## Features

- Interactive menu
- Display current date
- Display current user
- Display current working directory
- Exit option

---

## Technologies Used

- Bash
- Linux

---

## Skills Demonstrated

- Functions
- Loops
- Case Statements
- User Input
- Conditional Logic

---

## How to Run

```bash
chmod +x menu.sh
./menu.sh
```
