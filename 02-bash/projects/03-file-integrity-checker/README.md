# File Integrity Checker

## Overview

The File Integrity Checker generates MD5 and SHA256 checksums for a specified file.

This project demonstrates file validation, user input and hashing, which are commonly used to verify file integrity.

---

## Features

- Prompt for a file
- Validate file existence
- Generate MD5 hash
- Generate SHA256 hash
- Display results

---

## Technologies Used

- Bash
- Linux
- md5sum / md5
- sha256sum / shasum

---

## Skills Demonstrated

- User Input
- Conditional Statements
- File Validation
- Error Handling
- Command Substitution
