# User Account Creator

## Overview

The User Account Creator is a Bash script that automates the process of creating Linux user accounts.

The project demonstrates user input, conditional statements, functions and validation while simulating a common system administration task.

---

## Features

- Prompt for username
- Validate input
- Check if the user already exists
- Display success or error messages

---

## Technologies Used

- Bash
- Linux

---

## Skills Demonstrated

- Functions
- Variables
- User Input
- Conditional Statements
- Error Handling

## How to Run

```bash
chmod +x create-user.sh
./create-user.sh
```
